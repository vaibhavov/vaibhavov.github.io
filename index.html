About

I was fascinated by remote sensing satellite technology, the way they capture the world into something you can hold and reason about.

That fascination grew into a career at the edge of geospatial science and AI. My undergrad in Civil Engineering at IIT Roorkee gave me physical intuition; my master's at IIT Bombay gave me the tools to walk further, under Prof. Surya S. Durbha at CSRE, one of India's leading remote sensing labs.

I like working on problems which creates an impact in the real world. Like estimating crop yield, making autonomous agricultural systems. If I can replace some of that uncertainty with intelligent decision making, that's worth building for.

Outside research: I cook. Sometimes I sing. I like building things with my hands.

Projects

Things I've built

01
LangGraph GEE RAG Ollama M.Tech Thesis · IIT Bombay

GeoEye — an AI that thinks like an agronomist

A multi-agent framework that autonomously executes end-to-end multimodal data pipelines for real-time crop health assessment. Built on LangGraph with a shared AgentState schema across Master, Data, Monitoring, Yield, and Decision sub-agents — all running locally on an RTX A5000 with no cloud dependency. RAG pipeline built on ChromaDB and Qwen3 embeddings. Currently extending toward yield prediction and decision recommendation agents with self-learning capabilities and cross-regional knowledge transfer.

02
Prithvi 300M Sentinel-2 Fine-tuning TCS Research · May–Jul 2025

Mowing detection from orbit

Built a mowing detection pipeline for European grasslands to support sustainable biodiversity management. Fine-tuned the Prithvi 300M geospatial foundation model on Sentinel-2 time series data, reaching 87% accuracy. Built a custom dataset and end-to-end preprocessing framework from scratch, then ran a comparative state-of-the-art analysis to identify where geospatial foundation models still fall short.

03
PointNet++ LiDAR CUDA M.Tech Seminar · IIT Bombay

High-performance onboard processing for disaster mapping

Designed a PointNet++ architecture for real-time earthquake damage classification from LiDAR point clouds, with CUDA optimisations targeting deployment on Nvidia Jetson Nano. Reconstructed a 3D point cloud dataset from the Kumamoto Earthquake with a novel labelling strategy to support training. The constraint — embedded hardware with tight latency budgets — forced every design decision to be justified.

04
WGAN-GP EuroSAT GAN Self Project · Apr–May 2025

Generating satellite imagery that doesn't exist

Built a WGAN-GP to synthesise satellite images from the EuroSAT dataset across 10 land cover classes — useful for data augmentation when labeled imagery is scarce. The model learns spatial-textural features to generate 64×64 outputs. Validated generative quality on a downstream land cover classification task. Final metrics: FID 39.7, IS 4.3.

05
SwinIR Super-resolution Course Project · GNR 638

SwinIR transformer for 4× image super-resolution

Developed a SwinIR-based model with multi-head attention and residual connections to upscale images from 480×270 to 1920×1080 — a 4× factor. Optimised with patch-based training and L1 loss. Final performance: PSNR 24.8 dB, SSIM 0.76. Also built an HVAE for image reconstruction on the UC Merced Land Use dataset, improving representation learning through multi-level latent structure.

Code →
06
GCN Graph Learning Course Project · IE 506

Catching fake reviews with graph neural networks

Implemented fake review detection on the Yelp dataset using both supervised and unsupervised graph learning. Compared generation-based vs contrastive approaches across GCN architectures. Contrastive learning with GCN came out on top — AUC 0.79. The interesting finding was how much the graph structure itself carries signal that text features miss entirely.
